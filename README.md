**4 bit Arithmetic Logic Unit**  
---------------------------------------------------------------------------------------------------------------------------------
**Description**  
* 4 bit ALU is a logic operator that performs 4 bit addtion, subtraction using the logic gates, multiplexers and the output is shown
---------------------------------------------------------------------------------------------------------------------------------
**Features**
* List of operations  
      - Addition  
      - Subtraction  
      - AND,OR,NOT
---------------------------------------------------------------------------------------------------------------------------------
**Inputs & Outputs**  
* Inputs  
    - A (4-bit)  
    - B (4-bit)  
* Outputs  
    - Result (4-bit)
